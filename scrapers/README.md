# Scrapers

## Installing

For MacOS Users:
````
python3 -mvenv ./env
source ./env/bin/activate
pip3 install -r requirements.txt
````

For Windows Users:
````
pip3 install -U pip virtualenv
virtualenv --system-site-packages -p python3 ./venv
pip3 install -r requirements.txt
````

## Formatting code
Example:
````
yapf -i jetco.py
````

## Running the scraper
Example:
````
python3 jetco.py
````
